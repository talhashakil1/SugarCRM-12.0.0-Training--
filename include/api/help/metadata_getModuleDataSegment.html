Overview

[ADMIN] Gets the desired segment for a given module.

Summary

Gets the desired segment for a given module. This is used only for modules that are not visible and thus not in metatdata for the given context. This endpoint is only available to administrator.

Request Arguments

None

Response

{
    "fields": {
        Any fields associated with the module
    }
}
