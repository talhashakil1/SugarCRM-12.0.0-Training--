Overview

Retrieves the locale data for the user

Summary

This endpoint will return a set of locale settings, formatted as options, for the requested user.

Request Arguments

Name Type Description Required
actions Array Array of actions to be performed Yes

Request

    
        [
            {
                'type': 'CopyDashboards',
                'dashboards': ['id1', 'id2'],
                'modules': ['Accounts', 'Contacts'],
                'sourceUser': 'source-user-id',
                'destinationUsers': ['destinationUserId1', 'destinationUserId2'],
            },
            {
                'type': 'DeleteFilters',
                'filters': ['fid1', 'fid2'],
                'modules': [],
                'sourceUser': 'source-user-id',
                'destinationUsers': ['destinationUserId1', 'destinationUserId2'],
            }
        ]

Response

    
        true
