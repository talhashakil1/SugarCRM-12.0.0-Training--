Overview

Returns an installation status of a package installing by the Module Loader.

Request Arguments

Id
An uploaded package ID

Response Arguments

Error ErrorPage Message CurrentStep TotalSteps IsDone
An error description Boolean. True means - redirect to an error page fetched by an installation request. A progress message list. Contains some messages about a current progress step. Integer. Current progress step. Integer. Total progress steps. Boolean. True means that installation progress finished correctly.
