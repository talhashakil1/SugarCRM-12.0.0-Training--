Overview

[ADMIN] Gets metadata for Admin Panels.

Summary

This endpoint gets metadata for Admin Panels. This endpoint is only available to administrators and developers with access to any module.

Request Arguments

This endpoint does not accept any arguments.

Response

[
    'label' => 'LBL_TITLE',
    'description' => 'LBL_DESC',
    'options' => [
        [
            'label' => 'LBL_TITLE',
            'description' => 'LBL_DESC',
            'link' => 'link',
            'icon' => 'sicon',
            'customIcon' => 'icon.gif'
        ]
    ]
]
