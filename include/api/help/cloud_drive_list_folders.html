Overview

List folders from a cloud provider.

Request Arguments

Name Type Description Required
type String The cloud provider type(Google/OneDrive) Yes
folderId String The ID of the cloud folder No (default is root)
nextPageToken String Used for pagination No (default is null)
orderBy String used for ordering files/folders No
sharedWithMe bool Indicating if you want to list files from the shared drive or not No

Response

[
    [
        name: 'My folder',
        id: '1234',
        ...
    ]
    ...
]
