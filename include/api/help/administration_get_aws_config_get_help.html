Overview

[ADMIN] Gets Amazon Web Services configs from Sugar Serve.

Summary

This endpoint gets configuration values for Amazon Web Services. This endpoint is only available to administrators of Sugar Serve.

Request Arguments

This endpoint does not accept any arguments.

Response

{
    "aws_connect_instance_name": "my-aws-instance",
    "aws_connect_region": "us-east-2",
    "aws_connect_identity_provider": "SAML",
    "aws_login_url": "http://www.samllogin.com/",
    "aws_connect_enable_portal_chat": "1",
    "aws_logiaws_connect_api_gateway_urln_url": "http://123456789012.execute-api.us-east-2.amazonaws.com/Prod",
    "aws_connect_contact_flow_id": "2222",
    "aws_connect_instance_id": "1111",
}
