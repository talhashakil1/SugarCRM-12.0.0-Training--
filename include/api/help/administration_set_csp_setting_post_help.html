Overview

[ADMIN] Set Configuration Settings for Content Security Policy (CSP).

Summary

This endpoint sets configuration values Content Security Policy (CSP). This endpoint is only available to administrators.

Request Arguments

Name Value Description Required
csp_default_src String trusted domain names separated by space True

Response

{
    "csp_default_src": "google.com"
}
