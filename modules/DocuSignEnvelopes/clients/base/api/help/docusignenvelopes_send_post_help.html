Overview

Send an envelope

Summary

This endpoint will open a DocuSign tab to send an envelope. If the envelope needs to be created, it will create it, if the envelope was already created, it will only open it.

Request Arguments

Name Type Description Required
documents Array The list of documents that will be added to the envelope False
draftEnvelopeId String Envelope id already created in DocuSign False
returnUrlParams Object Parameters we need when action is finished in the new Docusign tab (eg. parentId, parentRecord, token) true

Response Arguments

Name Type Description
url String The url to DocuSign that needs to be accessed for the envelope to be configured and send/saved/discarded

Response

{
    url: "https://appdemo.docusign.com/auth-from-console?code=..."
}
