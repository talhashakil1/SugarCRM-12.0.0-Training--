Overview

Retrieve events to show in calendar

Request

POST /Calendar/getEvents

Response

The response will be an object containing the following:
Name Type Description
calendarConfigurations Array List of calendar configurations
endDate string End Date Event
location string The calendar location
startDate string Start Date Event

Response Example

{
    calendarConfigurations: [{
        "calendarId": "330cfcba-b7d2-11eb-8bf0-0242ac140004",
        "userId": "1",
        "teamId": ""
    }]
    endDate: "2021-08-07"
    location: "main"
    startDate: "2021-04-25"
}
