You can select from the following options to edit the SugarPortal:

  • Modül düzenle. Select this option to edit Bug Tracker, Cases, or Leads module.
  • Upload a Style Sheet. Select this option to edit the default SugarPortal stylesheet.
  • Portal Senkronizasyon. Select this option to synchronize SugarPortal with the Sugar server.

To edit a module, do the following:

  1. Select the module that you want to reconfigure in SugarPortal.
  2. Select the view that you want to edit.
    The Portal Editor displays a toolbar on the left and the detail view fields on the right. To change to Edit View or List View, select the appropriate link above. The toolbar displays the available fields that are currently not in the detail view. It also displays an empty field and a blank field. Use the blank field to create empty spaces in a view. For example, use a blank field next to Description to create adequate space to enter a detailed description. Use a empty field as a placeholder that you can replace with a field name in the future.
  3. To add a field from the Toolbar table to the detail view, click Add Row. Select the field from the toolbar and drag it to the empty field.
  4. To remove a field from the view, select it, and drag it to the Toolbar table.
  5. Click Save to save your changes.

To upload a style sheet, do the following:

  1. Select the appropriate tab for Cases, Bug Tracker, or Newsletters.
  2. Click Browse, navigate to the location of the style sheet, and select it.
  3. Click Upload to display the new style sheet.

To synchronize the SugarPortal with the Sugar server, do the following:

  1. Enter the URL to the portal instance that you want to update and click Go.
  2. Kullanıcı adınız ile şifrenizi girin ve tıklayınBegin Sync.
