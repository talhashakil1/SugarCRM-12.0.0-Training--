A list of forecasts entries matching filter criteria

Summary:

This endpoint is used to return the json Data for the committed values of a user's forecast based on the passed in filter criteria

Query Parameters:

Param Description Optional
user_id Show for a specific user, defaults to current user if not defined Optional
timeperiod_id Show for a specific time period, defaults to the current time period if one is not passed Optional
forecast_type Which forecast type to supply, rollup or direct required
show_deleted control if deleted committed forecasts should be included or not required

Input Example:

{ 'user_id':'seed_max_id, 'timeperiod_id':'36f7085a-5889-ea75-84c8-50f42bd1a5ba', 'forecast_type':'36f7085a-5889-ea75-84c8-50f42bd1a5ba', }
