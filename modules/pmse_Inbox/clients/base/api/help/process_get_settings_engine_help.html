Overview

Retrieve settings for the PA engine

Summary

This endpoint gets settings for the SugarBPMTM engine

Request Arguments

Name Type Description Required
fields string Comma separated list of SugarBPMTM config keys to return. If not sent, all config settings will be returned. false

Response Arguments

Name Type Description
logger_level string Level of logging
error_number_of_cycles integer Error number of cycles
error_timeout integer Timeout for errors

Response

{
    "logger_level":"critical",
    "error_number_of_cycles":"10",
    "error_timeout":"40"
}
